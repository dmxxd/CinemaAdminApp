# CinemaAdminApp

CinemaAdminApp — настольное приложение для автоматизации работы кинотеатра.
Проект предназначен для администраторов и сотрудников кинотеатра и
позволяет управлять основными бизнес-процессами в одном интерфейсе.

Проект выполнен в рамках учебной практики.

---

## Зачем нужен проект

CinemaAdminApp создан для упрощения и автоматизации рутинных операций
в кинотеатре, которые при ручном ведении занимают много времени
и подвержены ошибкам.

Приложение помогает:
- централизованно управлять фильмами и сеансами;
- контролировать продажу билетов;
- вести учет персонала и товаров буфета;
- формировать отчеты для администрации.

---

## Возможности

- добавление, редактирование и удаление фильмов;
- создание и управление сеансами без пересечений по времени в одном зале;
- продажа и регистрация билетов;
- управление персоналом и назначение смен;
- учет товаров буфета и контроль остатков;
- регистрация финансовых операций;
- формирование отчетов:
  - по продажам билетов;
  - по товарам буффета;
  - по сотрудникам;
  - по сеансам и операциям.

---

## Технологии

- Язык: C#
- Платформа: .NET
- Тип приложения: настольное (Windows)
- Среда разработки: Visual Studio Community 2022
- Интерфейс: Windows Forms / WPF

---

## Установка и запуск

1. Скачайте репозиторий или архив с проектом.
2. Распакуйте проект в удобную папку.
3. Откройте файл решения `CinemaARM.sln` в Visual Studio 2022.
4. Дождитесь восстановления зависимостей.
5. Запустите проект, нажав F5.

---

## Использование

После запуска приложения пользователь получает доступ
к административной панели, из которой можно:
- управлять фильмами и сеансами;
- продавать билеты;
- вести учет сотрудников и товаров;
- формировать отчеты за выбранный период.
